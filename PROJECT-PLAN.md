# Curbside Reference Website — Project Plan

## Goal
Build a work reference website that helps Hector quickly know where to go, what team he is on, and what location/building details matter for that day.

This project is meant to combine information for:
- Curbside 1
- Curbside 2
- Curbside 3

The idea is to create one clean place where Hector can check team-specific details without guessing or digging through scattered notes.

---

## Core Purpose
The website should answer questions like:
- What curbside team am I on today?
- Where do I report?
- What building/location is tied to that team?
- What entrance, parking, or access notes matter?
- What quick details do I need before starting work?
