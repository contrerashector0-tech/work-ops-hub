# Work Ops Hub

First-pass work dashboard for Hector.

## Current features
- Route sections for Curbside 1, Curbside 2, Curbside 3, and Parts Wash
- Quick links for work programs
- Mobile-friendly static site foundation
- Google Earth master map link
- Route-matched building panels using extracted Earth pin data where available
- Matched vs not-found building separation per route

## Current source status
Loaded:
- Curbside 1
- Curbside 2
- Curbside 3
- Parts Wash
- Google Earth SAA/NRAP location export

Current known gaps:
- Some workbook buildings still do not exist in the current Earth export, so they remain in "not found" lists
- Curbside 1 and Curbside 3 extracted files currently appear to contain the same building set and should be verified later
- More program link labeling
- Parking and reporting notes
